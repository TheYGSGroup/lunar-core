# YGS Lunar Core

Gregory Scott fork of [Lunar Core](https://github.com/lunarphp/core).

## Overview

This is a fork of `lunarphp/core` v1.1.0, customized for Gregory Scott projects.

**Package Name**: `ygs/lunar-core`  
**Base Version**: `1.1.0`  
**YGS Version**: `1.1.0-ygs1`  

## Installation

Add to your `composer.json`:

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "git@github.com:TheYGSGroup/lunar-core.git"
        }
    ],
    "require": {
        "ygs/lunar-core": "1.1.0-ygs1"
    }
}
```

## Versioning

- Base version matches upstream Lunar (e.g., `1.1.0`)
- YGS suffix indicates our fork version (e.g., `-ygs1`, `-ygs2`)
- Example: `1.1.0-ygs1` = upstream 1.1.0, first YGS release
